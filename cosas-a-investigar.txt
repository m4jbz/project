2. Clase SecureRandom y KeyGenerator
R: La clase SecureRandom se encarga
de generar los numeros aleatorios
adecuados para implementar en un
sistema de cifrado. Mientras que la
clase KeyGenerator se encarga de
generar las claves secretas a utilizar
en un algoritmo de cifrado
   en Java.

3. Que es SHA1PRNG.
R: Es un algoritmo de generacion de
números pseudoaleatorios.

== == == == == == == == == == == ==
     Como funciona el proyecto
== == == == == == == == == == == ==
Primero al hacer uso del metodo
pedirDatos() se escribe el contenido
dado por el usuario en archivos de
texto haciendo uso de la clase File.

Después sacas el contenido del
archivo.txt en bytes (dicho contenido
son las credenciales dadas por el
usuario en forma de texto), cifras
el contenido con la LLAVE generada
a partir de las clases SecureRandom,
Cipher y KeyGenerator, colocas el
contenido cifrado en un nuevo en
bytes dentro de un nuevo archivo.bin.

Para poder acceder a el contenido de
dicho archivo se realiza un proceso
inverso, se obtiene primero los bytes
del contenido encriptado se
desencripta con la LLAVE y con la
clase Cipher para luego (si es pedido)
mostrar el contenido desencriptado en
pantalla.
